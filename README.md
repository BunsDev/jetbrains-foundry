# intellij-foundry
![](media/logo.png)

#### Run [foundry](https://github.com/foundry-rs/foundry) commands within IntelliJ IDEA, simply click the button.

![](media/screenshot.gif)

### Development
```shell
# Build gradle project
./gradlew clean && ./gradlew buildPlugin

# Run plugin
./gradlew runIde
```
